
# FINM3422 – Financial Modelling & Analytics for Finance (Course Repo)

This repository contains **lecture and tutorial materials** for FINM3422 at The University of Queensland

## What this repo is

- Jupyter notebooks used in **lectures** (examples, live demos)
- Starter notebooks and scaffolds for **tutorials**
- Small sample **datasets** for practice
- An example **Conda environment** (`env/environment.yml`)

Students are encouraged to **clone this repo and run the notebooks locally** using VS Code + Conda.

## What this repo is NOT

- It is **not** your Assessments repository.
- It does **not** contain complete solutions for any assessment.
- You must **not fork this repo and submit it** as your own work.

For Assessments, each team must create their **own private GitHub repo**, with:
- their own folder structure
- their own code
- appropriate documentation of any AI/tool assistance
